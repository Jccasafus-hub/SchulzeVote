Método de Schulze

O método de Schulze é um sistema de votação por comparação par-a-par (um método de Condorcet) usado para determinar um ranking entre candidatos ou opções. Ele é utilizado em diversas organizações, incluindo a Debian Foundation e a Wikimedia Foundation.

1. Coleta das preferências

Cada eleitor fornece uma ordem de preferência entre os candidatos. Exemplo com quatro candidatos (A, B, C, D): 

Eleitor 1: A > C > B > D
Eleitor 2: B > A > D > C
Eleitor 3: C > D > B > A
Eleitor 4: A > B > C > D

2. Matriz de preferências par-a-par

Para cada par (X,Y), contamos quantos eleitores preferem X sobre Y. Isso gera a matriz P[X][Y].

Exemplo parcial:
  • P[A][B] = 3 (3 eleitores preferem A a B)
  • P[B][A] = 1 (1 eleitor prefere B a A)
  • P[C][D] = 2, etc.

3. Cálculo dos caminhos mais fortes

Para cada par (X,Y), calculamos a força do caminho mais forte de X até Y, onde a força de um caminho é definida como o menor valor de vitória ao longo dele. Esse cálculo é feito com um algoritmo de propagação do tipo Floyd–Warshall.

4. Comparação final

Dizemos que X é preferido a Y se S[X][Y] > S[Y][X]. O ranking final é construído ordenando os candidatos por essas comparações.

Exemplo numérico

   Matriz de vitórias (P):
       A   B   C   D
    A  -   3   2   4
    B  1   -   3   2
    C  2   1   -   3
    D  0   2   1   -

   Caminhos fortes (S):
       A   B   C   D
    A  -   3   2   3
    B  1   -   2   2
    C  2   2   -   2
    D  1   2   1   -

Comparando S[X][Y] com S[Y][X], obtemos a ordem final: A > B > C > D.

Visualização em diagrama

       A
      ↗ ↘
     B   C
      ↘ ↗
       D

As setas representam as vitórias mais fortes em cada comparação.

Vantagens

  • Identifica o vencedor de Condorcet, se existir.
  • Produz um ranking completo, não apenas um único vencedor.
  • É resistente a clones (candidatos semelhantes não prejudicam uns aos outros).
  • Já foi amplamente testado e usado em eleições reais.

Limitações

  • Pode ser mais difícil de explicar ao público em geral.
  • Exige cálculos matriciais, não sendo prático para apurações manuais.
